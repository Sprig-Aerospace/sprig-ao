# Sprig workspace config docs

This directory holds workspace-level configuration artifacts that describe how the Sprig repositories fit together.

## Files

- `sprig-workspace.manifest.yaml` — initial workspace manifest for Milestone 1: workspace awareness.

## Notes

- Repository roles and ownership notes are based on the repository descriptions and READMEs available on **May 12, 2026**.
- `sprig-sitl-workspace` is included as a planned workspace entry because no GitHub repository with that exact name was discoverable in the `Sprig-Aerospace` organization on **May 12, 2026**.
- Runtime profiles are intentionally high-level so the manifest can guide orchestration decisions without duplicating every repo-specific launch script.
